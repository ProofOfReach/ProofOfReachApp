Getting Started

1
Add the SDK Script

Add this line to the bottom of your HTML page, just before the closing </body> tag:

<script src="https://cdn.proofofreach.com/sdk/v1.js"></script>
2
Create an Ad Container

Add this div element wherever you want the ad to appear:

<div id="proof-of-reach-ad"></div>
3
Initialize the SDK

Add this script after the SDK import:

<script>
  window.addEventListener("load", () => {
    ProofOfReachSDK.renderAd("proof-of-reach-ad", {
      pubkey: "YOUR_NOSTR_PUBKEY", // Your Nostr public key
      testMode: false              // Set to true for testing
    });
  });
</script>
